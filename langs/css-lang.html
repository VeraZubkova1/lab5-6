Краткое описание

CSS — позволяет задавать цвета, шрифты, размеры, расположение элементов и создавать анимации.

Основные возможности CSS - это:

  • Управление цветами и шрифтами
  • Создание адаптивного дизайна
  • Анимация элементов
  • Расположение элементов на странице

Пример исходного кода


body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    margin: 0;
    padding: 20px;
}

h1 {
    color: #333;
    text-align: center;
}

.button {
    background-color: #007bff;
    color: white;
    padding: 10px 20px;
    border-radius: 5px;
}

.button:hover {
    background-color: #0056b3;
}
