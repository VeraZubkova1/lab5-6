Краткое описание

HTML — используется для создания веб-страниц. С помощью HTML-тегов можно задавать структуру документа: заголовки, абзацы, списки, таблицы, изображения и многое другое.

Основные возможности HTML - это:

  • Создание структуры веб-страницы
  • Вставка изображений, видео и аудио
  • Создание гиперссылок для навигации
  • Работа с формами для ввода данных

Пример исходного кода

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя страница</title>
</head>
<body>
    <h1>Заголовок</h1>
    <p>Это абзац текста.</p>
    <a href="https://example.com">Ссылка</a>
</body>
</html>
